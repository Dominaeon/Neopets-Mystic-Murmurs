===============================================

### Description
This is a solver for the mystic murmurs puzzle for the novemeber plot in neo. Just replace the text in text1,2,3 .txt with copy pasts from your murmer. 

The answer will be one of these locations:

    Anchor Management
    Exquisite Ambrosia
    Haunted House
    Slushie Shop
    Turdle Racing
    Qasalan Delights
    Ugga Shinies
